# Suggested Commands
- `npm install` – install dependencies.
- `npm run dev` – start Vite dev server at http://localhost:5173.
- `npm run build` – produce production bundle in `docs/`.
- `npm run preview` – serve the built bundle for smoke testing.
- `npm test` – run full test suite (unit + E2E where configured).
- `npm run test:unit` – execute 227-unit test suite quickly during refactors.
- `npm run test:coverage` / `npm run test:coverage:check` – generate and enforce coverage.
- `npm run clean` – clear build artifacts before reproducible builds.
- `npm run help` – list all available npm scripts.
- `uvx codetoprompt: compress …` – regenerate `llms.txt` snapshot per workflow guidance.
